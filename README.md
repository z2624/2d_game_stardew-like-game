# 2D Game - Stardew-like

一个类似星露谷物语的 2D 农场模拟游戏项目。

## 技术栈
- **引擎**: Godot 4.x
- **语言**: GDScript
- **美术风格**: 像素艺术

## 目录结构
```
.
├── src/              # 源代码
├── scenes/           # Godot 场景文件
├── assets/           # 资源文件
│   ├── sprites/      # 精灵图
│   ├── audio/        # 音频
│   └── fonts/        # 字体
├── docs/             # 文档
└── README.md         # 本文件
```
